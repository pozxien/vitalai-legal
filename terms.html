Global Features Policy

Global Chat

Global Chat connects your server to a cross-server messaging network. Messages are relayed via webhooks and are not stored by Vital AI.

Rules:

  • No invite links
  • No @everyone/@here mentions
  • No hate speech, slurs, or harassment
  • No NSFW content
  • 2 second cooldown between messages

Enforcement: Violations result in automatic permanent ban from Global Chat. Messages are relayed with your username and server name visible to all connected servers.

Global Advertising

Global Ads allows servers to broadcast advertisements to other participating servers.

Rules:

  • No hate speech or discriminatory content
  • No NSFW server advertisements
  • No scam or phishing servers
  • Cooldowns: Free (4h), Starter (2h), Pro (1h)

Visibility: Your server name, member count, and user ID are visible when posting ads. Pings are stripped from all relayed content.

Opting Out

To stop participating in Global features:

  • Global Chat: Use /global disconnect
  • Global Ads: Use /ads disable

Your server will immediately stop sending and receiving global messages.

Ban Appeals

If you believe you were wrongly banned from Global features, contact us on our support server with:

  • Your User ID
  • Server ID (if applicable)
  • Explanation of the situation

Appeals are reviewed within 7 days. Decisions are final.
